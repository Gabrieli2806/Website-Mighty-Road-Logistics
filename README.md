## 🚀 How to Use

1. **Open the website**: Simply open `index.html` in a web browser
2. **Customize images**: Replace logo1.jpg, logo2.jpg, logo3.jpg with your actual logos
3. **Update content**: Edit text in index.html as needed
4. **Adjust colors**: Modify CSS variables in styles.css root section
5. **Add social links**: Update href attributes in footer social links

## 🎨 Color Customization

To change colors, edit the CSS variables in `styles.css`:

```css
:root {
    --primary-gold: #D4AF37;    /* Main gold color */
    --dark-gold: #B8960F;       /* Darker gold for hover */
    --black: #0A0A0A;           /* Main black */
    --white: #FFFFFF;           /* Pure white */
}
```
